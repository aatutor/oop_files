1) Реализовать классы "точка" и "круг" агрегированием. 
    - Точка имеет координаты (х, у). 
    - Круг имеет поля класса точки (агрегирование) и радиус.
    Реализовать в классах: 
    - методы для вывода информации об объектах
    - методы для перемещения точки и окружности

    Протестировать работу методов с помощью объектов обоих классов.

2) Реализовать классы из №1 с помощью наследования. Избавиться от лишних методов.

    Протестировать работу методов с помощью объектов обоих классов,
    автоматических и динамических переменных, массива РАЗНЫХ объектов.

3) Реализовать класс "пиксель", производный (наследник) от класса "точка".
    - Пиксель имеет поле с цветом (char).
    Реализовать в классе:
    - метод изменения цвета
    - метод вывода информации

    Протестировать работу с объектами.

    3.1*) Добавить класс "дисплей", содержащий матрицу пикселей и массив точек.
    Массив точек (указателей) может быть набором производных классов. 
        Например: круг, треугольник, квадрат.

4) Реализовать классы "Student", "Teacher" и "Person".
    Первые классы наследуются от "Peson". 
    - Person имеет поля с именем и возрастом.
    - Student имеет группу и оценки по предметам.
    - Teacher имеет предмет.
    Реализовать в классах:
    - инициализация всех полей объектов (конструкторы и методы)
        - имя, возраст, группа, предмет
    - метод вывода информации о человеке
    - метод увеличения возраста

    Протестировать работу с объектами.
